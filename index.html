> Qu'est ce que Git ?

Git est un logiciel de contrôle de version utilisé pour le développement de projet, souvent logiciel. Les choses suivant sont les principales fonctionnalités de Git :

  • Les commits sont les mise à jour des fichiers modifiés depuis le dernier commit. Ce sont les unités de sauvegarde de git.
  • Les branchs sont des versions divergentes du même projet, par exemple pour testés des prototypes d'algorithme.
Les commits sont l'ensemble des modifications effectués sur un projet. Les commit sont classés par date (et se voient attribués un numéro de révision unique) et sont une suite incrémentale des changements appliqués sur le projet (ajout, modification ,suppression de fichiers). Tout est gardé et récupérable qu'elle que soient les modifications apportées entre temps.Les commits sont fait avec la commande commit.
